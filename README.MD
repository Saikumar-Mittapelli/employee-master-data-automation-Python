# Employee Master Data Automation (Python)

This project automates employee data processing using Python and Pandas.

## Features
- Merges employee data with email address lookup
- Calculates annual salary from monthly salary
- Computes employee tenure in months
- Categorizes employees into tenure groups
- Generates Employee Master Data file
- Automatically splits data by department

## Technologies Used
- Python
- Pandas
- Excel (openpyxl)

## Use Case
This automation helps HR and analytics teams reduce manual Excel work and ensures consistent reporting.
